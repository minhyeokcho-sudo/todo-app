      GitHub Gist 동기화 설정

      본인의 GitHub 계정에 비공개 Gist를 만들어 여러 기기에서 같은 할 일 목록을 사용할 수 있게 합니다. 한 번만 설정하면 됩니다.

      1. → GitHub 토큰 발급 페이지 열기
        scope는 gist만 체크, 만료 기간은 본인 선호대로
      2. 발급된 토큰(ghp_로 시작)을 복사
      3. 아래에 붙여넣고 저장하고 동기화 클릭

      토큰은 이 브라우저의 localStorage에만 저장됩니다. gist 권한 외에는 부여하지 않으니 다른 리포·이슈에는 접근할 수 없습니다. 변경하려면 동기화 버튼을 우클릭하세요.
